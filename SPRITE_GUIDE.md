# 도트 스프라이트 제작 가이드

사용자가 직접 도트를 찍어 게임의 placeholder 그래픽을 교체하기 위한 규격 문서입니다.
파일을 만들어 전달하면 로딩 코드 연결과 `ASSET_CREDITS.md` 기록(자작 표기)은 Claude가 처리합니다.

## 공통 규칙

- **PNG, 투명 배경** (배경색 채우지 않기)
- **원본 크기 그대로** 제작 — 게임이 알아서 확대합니다. 미리 키워서 저장하지 않기
- **안티앨리어싱 금지** (부드럽게 번지는 브러시 대신 1px 연필 도구)
- 어두운 바닥(#121a22 근처) 위에 올려보고 잘 보이는지 확인
- 파일명은 소문자와 하이픈 (예: `root-gnarl.png`)
- 전달 방법: `tmp/sprites/` 폴더에 넣고 Claude에게 알려주기 (tmp는 git에 안 올라가는 작업용 폴더)

## 제작 팁

- **실루엣 우선**: 색을 빼고 검은 그림자만 봐도 어떤 적인지 구분되면 성공
- **1px 어두운 외곽선**을 두르면 어두운 바닥에서도 또렷해짐
- **명암 3단계** (어두움·기본·밝음)면 충분함. 광원은 좌상단 기준
- 애니메이션은 선택 사항: 우선 1장(정지)으로 시작하고, 여유가 되면 2~4프레임
  (idle 숨쉬기 등)을 가로로 이어 붙인 시트로 — 연결은 Claude가 처리
